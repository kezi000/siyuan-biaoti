# Changelog

## v0.5.0 2025-11-24

* Introduce AI Title Assistant with OpenAI-compatible configuration UI
* Add manual toolbar button/command that sends context to the configured LLM and renames the current document
* Support custom prompt templates, tone presets, and context strategies
* Support multiple AI providers: OpenAI, Anthropic, Gemini, DeepSeek, and Zhipu AI
* Add fallback provider mechanism for improved reliability
* Implement comprehensive error handling and retry policies
* Add support for multiple languages and tone presets
* Include context strategy options (auto, selection, block, document)
* Provide real-time connection testing for API endpoints
